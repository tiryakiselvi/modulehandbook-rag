# Outputs

Dieser Ordner enthält dokumentierte Beispielausgaben der Retrieval-Evaluation. Die Dateien können mit den Befehlen aus dem README neu erzeugt werden, sobald die Chunks unter `data/processed/` erstellt wurden.

`final_evaluation/` enthält die abschließenden maschinenlesbaren Ergebnisse: Gesamtmetriken, Auswertung nach Fragetyp, Per-Query-Fehleranalyse, Chunk-Integrität und den Pilot zum Antwortverhalten.
