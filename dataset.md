# Dataset notes

The repository does not redistribute LMU module handbooks or generated chunk files. Before running an experiment, obtain the relevant document from its official source and place it in `data/raw/`.

For the pilot reported in `RESULTS_2026-06-23.md`, the input was the LMU Bachelor Computerlinguistics module handbook. Generated JSONL chunk files are intentionally local artefacts and can be recreated with the ingestion commands in the README.
